# دليل النشر الدائم - Easy Al Obour

## 🌐 خيارات النشر الدائم المتاحة:

### 1. Vercel (الموصى به - مجاني وسهل)
**المميزات:**
- ✅ نطاق مجاني: `easy-al-obour.vercel.app`
- ✅ SSL/HTTPS مجاني
- ✅ CDN عالمي
- ✅ نشر تلقائي من GitHub
- ✅ سهل الاستخدام

**الخطوات:**

```bash
# 1. تثبيت Vercel CLI
npm install -g vercel

# 2. تسجيل الدخول
vercel login

# 3. نشر التطبيق
vercel --prod

# 4. إضافة متغيرات البيئة
vercel env add JWT_SECRET
vercel env add DATABASE_URL (اختياري)

# 5. إعادة النشر
vercel --prod
```

**الرابط الدائم:**
```
https://easy-al-obour.vercel.app
```

## 🔧 إعداد النطاق المخصص (Domain)

### شراء نطاق:
1. اذهب إلى https://namecheap.com أو https://godaddy.com
2. ابحث عن النطاق المطلوب (مثل: `easy-al-obour.com`)
3. اشترِ النطاق

### ربط النطاق بـ Vercel:
```bash
# 1. في لوحة تحكم Vercel
# 2. اذهب إلى Project Settings > Domains
# 3. أضف النطاق الجديد
# 4. اتبع التعليمات لتحديث DNS
```

### ربط النطاق بـ Railway/Render:
```bash
# 1. في لوحة تحكم الخدمة
# 2. اذهب إلى Custom Domain
# 3. أضف النطاق
# 4. حدّث سجلات DNS عند مزود النطاق
```
